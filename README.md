# Maya Landscaping — Sitio Web

Sitio web profesional para Maya Landscaping en Houston.

## 🚀 Despliegue en Netlify

### Opción 1: Despliegue automático (recomendado)

1. **Conecta tu repositorio GitHub a Netlify:**
   - Ve a [netlify.com](https://netlify.com)
   - Haz clic en "New site from Git"
   - Conecta tu cuenta de GitHub
   - Selecciona este repositorio
   - Netlify detectará automáticamente la carpeta `deploy/`

2. **Configuración automática:**
   - **Build command:** (déjalo vacío — no se necesita compilación)
   - **Publish directory:** `deploy/`

3. **¡Listo!** Tu sitio estará en vivo en minutos.

### Opción 2: Despliegue manual

1. Ve a [netlify.com](https://netlify.com)
2. Arrastra la carpeta `deploy/` al área de drop
3. ¡Tu sitio estará publicado inmediatamente!
